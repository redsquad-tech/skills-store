# Setup Bundle

## Purpose
Initializes and verifies the business-md-bundle environment.

This skill ensures that all required tools and dependencies are installed and the vault is properly initialized.

---

## Capabilities

- Install required tools
- Verify environment
- Prepare vault structure

---

## Commands

- bundle-install ensure core
- bundle-install ensure qmd
- bundle-install ensure markitdown
- bundle-install verify

---

## Behavior

- Does not modify user data
- Ensures environment readiness

---

## Expected Outcome

- Environment is ready
- All required tools are available
- Vault structure exists
